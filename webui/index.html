teibo

堤体すべり安定計算 / ブラウザ内完結
「照査実行」を押すと、臨界すべり円の探索結果がここに表示されます。
対応: 円弧すべり（修正フェレニウス法/簡易ビショップ法）・上載荷重・外水位・浸潤線自動推定（カサグランデ）・ テンションクラック・探索拘束・液状化(FL法)・ニューマーク法（滑動変位量）・ 対策工の比較（押え盛土・地盤改良・ドレーン）・感度分析・入力診断・帳票出力。
入出力はteibo CLIと同じJSON形式のため、このシートで作った入力はそのまま python -m teibo analyze に渡せます。
ブラウザの印刷ダイアログで「PDFに保存」を選ぶとPDFになります。
複数断面の一括照査（縦断） 距離標ごとの断面をまとめて照査し、縦断方向のFsを一覧・グラフ表示します。
